# StChurch Website

A church website developed using **HTML, CSS, and PHP**.
The website provides information about church events, ministries, galleries, and newsletters. It also includes an **admin panel** for managing website content.

## Technologies Used

Frontend

* HTML5
* CSS3

Backend

* PHP

Server Environment

* Apache (XAMPP / WAMP recommended)

## Project Features

* Church home page
* Events listing and event details
* Church ministry information
* Image gallery
* Newsletter section with downloadable PDF newsletters
* Admin panel for content management

## Project Structure

```
StChurch
│
├── admin/                # Admin panel for managing content
├── image/                # Website images and gallery images
├── newsletters/          # Newsletter PDF files
│
├── Home_page.html        # Website home page
├── events.html           # Events listing page
├── event-details.html    # Event details page
├── gallery.html          # Image gallery page
├── ministry-details.html # Ministry information page
├── newsletter.html       # Newsletter page
├── newsletter.css        # Newsletter styling
```

## Installation

### 1 Clone the repository

```
git clone https://github.com/affanm945/StChurch.git
```

### 2 Move project to web server

Place the project folder inside your web server directory.

Example for XAMPP

```
C:\xampp\htdocs\
```

### 3 Start Apache Server

Open **XAMPP Control Panel** and start:

* Apache

### 4 Run the Website

Open browser and navigate to

```
http://localhost/Church
```
